# 海外利益保护风险监测预警系统

> 面向中资企业海外利益保护的风险监测、预警与评估平台

## 功能模块

| 模块 | 说明 |
|------|------|
| 综合仪表盘 | 全球风险概览、关键指标、最新预警 |
| 风险地图 | SVG 世界地图、航运通道、一带一路走廊、区域雷达 |
| 企业监测 | 35 家中资企业风险画像与详情 |
| 国家监测 | 42 国风险分布、区域对比、趋势摘要 |
| 预警中心 | 30 条实时预警、4 级响应、处置流程 |
| 事件追踪 | 25 起重大安全事件时间线 |
| 人员安全 | 人员安全事件、撤离追踪、应急联络 |
| 预测中心 | 3/6/12 月预测、情景分析、早期预警指标 |
| 风险评估 | 8 维度 32 指标加权评分模型 |
| 风险矩阵 | 企业×国家象限分析、典型案例 |
| 统计分析 | 行业×区域热力图、预警交叉分析 |
| 评估报告 | 自动生成风险报告 |
| 系统设置 | 预警阈值、响应级别配置 |

## 技术栈

- 纯前端 HTML + JavaScript（无后端依赖）
- Chart.js v4.4.1 数据可视化
- SVG 世界地图与自定义图表
- 响应式布局

## 本地运行

```bash
# 方式 1：Python
python -m http.server 8080

# 方式 2：Node.js
npx serve

# 然后打开浏览器访问 http://localhost:8080
```

## 部署

直接将 `index.html` 和 `app.js` 部署到任意静态文件服务器即可。
